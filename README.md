# 📝 Scribo Blog App – Full Stack Blog Platform  

## Project Overview  
Scribo Blog App is a **full-stack blogging platform** built with **Next.js** that includes a **frontend, backend, and admin panel**.  
It provides a seamless experience for creating, managing, and reading blog posts with **responsive UI** and smooth navigation.  

---

## Tech Stack  

**Frontend & Backend:**  
- Next.js (React Framework)  
- REST APIs (built within Next.js)  

**Database:**  
- MongoDB (with Mongoose ODM)  

**Other Tools & Libraries:**  
- Git/GitHub (Version Control)  
- TailwindCSS / CSS (for styling)  
- Node.js Runtime  

---

## ✨ Features  
-  **Create, Read, Update, Delete (CRUD)** operations for blog posts.  
-  **Admin Panel** – Manage blog posts and content easily.  
-  **Dynamic Frontend** – Blog posts displayed with smooth navigation.  
-  **Responsive Design** – Works across desktops, tablets, and mobiles.  
-  **API Integration** – Connected blog posts with a MongoDB database.
